# Archive

This directory contains archived reference files from the legacy prototype.
These files are **not** part of the active build or deployment pipeline.

## Contents

- **`build-single-file.mjs`** — Legacy single-file build script (HTML-in-one)
- **`peacefull-demo-github/`** — Legacy static demo site for GitHub Pages
- **`prototype/`** — Legacy prototype JS files from `prototype-web/public/js/`

## Why Keep Them?

Historical reference for how the project evolved from a monolithic HTML/JS
prototype to the current React + TypeScript SPA with Express API backend.

> ⚠️ Do not import, reference, or depend on any files in this directory.
